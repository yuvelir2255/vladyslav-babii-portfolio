# /audio

Сюда положить фоновый эмбиент для тумблера звука (`AudioToggle`).

- **Файл:** `ambient.mp3` (именно этот путь, `/audio/ambient.mp3` — код менять не нужно).
- **Лицензия:** только royalty-free / CC0 (тюремный / индустриальный эмбиент).
- **Поведение:** off по умолчанию; включается кликом, громкость 0.4, луп.
- Пока файла нет — кнопка работает «вхолостую» (`play()` отклоняется, остаёмся off), сайт не ломается.

> Этот README — заглушка-указатель. После добавления `ambient.mp3` его можно удалить.
