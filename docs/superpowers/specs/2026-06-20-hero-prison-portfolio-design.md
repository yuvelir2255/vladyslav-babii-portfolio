# Дизайн-спека: портфолио «Case File» (концепт + дизайн-система + Hero)

> Дата: 2026-06-20. Статус: согласовано владельцем (дизайн hero утверждён в брейншторме).
> Это спека ПЕРВОГО среза (фундамент + hero). Остальные секции — отдельными спеками позже.
> Английские строки-копирайт оставлены как есть (язык сайта — EN; UA-локаль позже).

## 1. Концепт и позиционирование

**Идея:** портфолио в эстетике тюрьмы / «дела» (case file), брутализм-бетон + сталь + prison-orange.
Метафора-флекс: **тюрьма = система/шаблон/демо-клетка, а Vladyslav — тот, кто из неё сбегает.**
«Срок» = опыт, «номер» = dev-ID, «обвинение» = флекс (`shipping real products`), `AT LARGE` = доступен.
Подтекст — «неудержимый билдер», НЕ «преступник».

**Аудитория:** обычные люди — познакомиться с автором (кто он, что умеет). Не рекрутёр-фильтр.
Внизу сайта — мягкая заявка для агентств/предпринимателей. Поэтому максимализм и дерзость уместны.

**Цель сайта:** макс. вау-эффект, вовлечение (хочется тыкать, скроллить, взаимодействовать).

## 2. Дизайн-система

### Цвет (тёмный бетон)
- `--bg` `#100f0d` — фон (глубокий бетон)
- `--bone` `#ece7da` — основной светлый текст
- `--steel` `#9c968a` — вторичный текст
- `--dim` `#6f6a60` — третичный/подписи
- `--orange` `#ff5a1e` — единственный акцент (prison-orange)
- `--line` `rgba(236,231,218,.12)` — разделители/бордеры
- Фон-зерно (SVG feTurbulence overlay), едва заметные «прутья» (repeating-linear-gradient).

### Типографика
- **Anton** — дисплей (имя, манифест-заголовки). Крупно, condensed, постерно.
- **JetBrains Mono** — системный текст, тикер, метаданные, подписи, кнопки.
- **Oswald** — «настоящая» booking-табличка и лейблы (push-letter вид).
- **Saira Stencil One** — трафаретные штампы (`AT LARGE`, `ACCESS GRANTED`).
- Грузить через `next/font` (Google), без layout-shift.

### Движение (важно)
- **prefers-reduced-motion НЕ уважаем** — решение владельца: анимация всегда включена для всех.
  (В спеке зафиксировано осознанно; обычная a11y-страховка снята намеренно.)
- Максимализм, но курируемо: каждый эффект живёт в мире «бетон/дело/побег», цель **60fps**,
  тяжёлое грузим лениво, на мобилке — облегчённые версии (ради перфа, не ради reduced-motion).

## 3. Стек и библиотеки (согласованный арсенал)

- База: **Next.js 16 (App Router) · React 19 · TS strict · Tailwind v4** (токены в `globals.css` через `@theme`).
- Движение: **GSAP** (бесплатно, все плагины) — ScrollTrigger, **SplitText**, **ScrambleText**, **DrawSVG**, **Flip**.
- Скролл: **Lenis** (плавный инерционный).
- Шейдер-фон: **OGL** (лёгкий WebGL — бетон-туман + луч прожектора + сканлайны, реакция на курсор) — Tier A.
  (Тяжёлый R3F/3D-сцена — НЕ берём, владелец отказался от совмещения тиров.)
- Курсор: **кастомный** (прожектор/прицел-scan, магнитные кнопки, «декод» текста под курсором).
- Картинки: шейдер-эффекты на hover (displacement/RGB-split/«цензура»).
- Физика: **Matter.js** (перетаскиваемые «улики/ключи») — позже, не в hero.
- Переходы: **View Transitions** (между секциями — «сдвиг двери камеры») — позже.
- Звук: маленький тумблер, **выкл по умолчанию**, royalty-free/CC0 тюремный эмбиент-луп.
- Ассеты: генерация через **Higgsfield** (Nano Banana Pro для лица, и т.п.).

## 4. Hero (первый срез сборки = фундамент + hero)

### 4.0 Прелоадер «lockdown»
- Полноэкранный INTAKE TERMINAL: счётчик `0→100`, скан-линия, расшифровка «дела»
  (`scanning biometrics… / matching record… MATCH / charge: shipping real products / clearance… GRANTED`),
  прогресс-бар, штамп `ACCESS GRANTED`. На 100% «дверь камеры» уезжает и открывает hero.
- Заодно прячет загрузку ассетов (шрифты, мугшот, шейдер).

### 4.1 Экран 1 — удар (≈100vh)
- **Тикер** сверху (PA-объявление, бежит):
  `INTAKE TERMINAL — BLOCK VB // STATUS: AT LARGE // CHARGE: SHIPPING REAL PRODUCTS // NOW SHIPPING FROM WARSAW // VISITING HOURS: ALWAYS OPEN //`
- **Слева (воздух, легко читается):**
  - Мета: `CASE FILE №VB-19 · KHARKIV → WARSAW · AT LARGE`
  - **Имя** `Vladyslav / Babii` — Anton, крупно, с дыханием (увеличенный leading).
    Анимация: буквы «штампуются» (SplitText) на входе; **глитч/сканлайн на hover**.
  - **Роль** (одна строка, без «Developer»): `Telegram Mini Apps · Websites · AI Products`
  - **CTA:** `Open the case file ↗` (→ кейс Dream Gold) · `Make contact · visiting hours open` (→ заявка)
- **Справа (крупный мугшот):**
  - Booking-фото (мугшот v2, плейсхолдер; позже Soul-версия — просто замена файла).
  - Тег `EXHIBIT A`, штамп `AT LARGE`.
  - **Табличка — настоящий letterboard, текст в DOM** (чёткий, анимируемый «декодом»):
    `DEPARTMENT OF SHIPPING / UKRAINE / 06·20·2026   VB-19`
- **Фон:** OGL-шейдер бетона (туман + прожектор + сканлайны), реакция на курсор.
- **Курсор:** прожектор/scan.
- Низ: `scroll to escape ↓`.

### 4.2 Экран 2 — манифест (≈82vh, scroll target)
- Лейбл `ON THE RECORD / STATEMENT 01`.
- Манифест крупно (Anton), оранжевые ключи, **ScrambleText-декод построчно по скроллу**:
  > I don't write demos. I build products that **escape the lab** and take **real orders**.
  > Whatever I start — I **break out** with it finished.
- Переход экран1→экран2 = часть нарратива «scroll = escape».

### 4.3 Адаптив
- Mobile-first; проверка 375 / 768 / 1280; без горизонтального скролла; тач-цели ≥44px.
- Мобилка: колонки в стек (мугшот под именем/над), размер шрифта через `clamp()`, тикер сохраняется,
  шейдер — облегчённый/статический фолбэк ради перфа.

### 4.4 a11y (с учётом решения по движению)
- reduced-motion намеренно НЕ уважаем (см. §2).
- Прочая a11y сохраняется: контраст ≥4.5:1 для основного текста, видимый `:focus-visible`,
  `alt` у мугшота, `aria-label` у иконок-кнопок (звук-тумблер), навигация с клавиатуры.
